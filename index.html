Why?

  • Enjoy js libs from a new point of view
  • One environment, don't repeat yourself
  • Gradle, Grails, Spock, Codenarc, ...
  • Type checking if you need it
  • @Ast's, Dsl's, scripts, traits, beans, ...
  • Just Groovy, use js when you want to
