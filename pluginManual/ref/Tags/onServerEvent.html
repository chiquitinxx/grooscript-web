  • Quick Reference

Use your groovy code in your gsps thanks to GrooScript. It converts the code to javascript and your groovy code will run in your browser. Also use Vert.x to use events between server and gsps.

(Quick Reference)

onServerEvent

Code that will runs when a Vert.x event in that channel triggers. message is param available in the code.

<grooscript:onServerEvent name='nameChannel'>
    console.log 'Message: ' + message
</grooscript:onServerEvent>

name - required - name of the channel where listening.
